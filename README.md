# Grocery POS System

## 📖 Introduction
The Grocery POS (Point of Sale) System is a lightweight application designed to simplify billing and inventory management for small grocery shops. Built with **Node.js** and a simple JSON-based backend, it provides a clean and efficient way to handle day-to-day transactions without requiring complex database setups.

## ✨ Key Highlights
- 🛒 Manage product catalog (add, edit, delete items)  
- 💵 Generate customer bills with itemized details  
- 📊 Track and store sales records  
- ⚡ Fast and easy to run locally  
- 🔧 Simple codebase, easy to extend with MySQL or other databases  

## 🛠️ Installation
Clone the repository:
```bash
git clone https://github.com/Hyder9-flash/Grocery-pos.git
cd Grocery-pos

Install dependencies:
npm install

▶️ Usage
Start the server:
node server.js
Open index.html in your browser to interact with the system.
📂 Project Structure
Grocery-pos/
│── index.html        # Frontend interface
│── server.js         # Backend server logic
│── products.json     # Product catalog storage
│── sales.json        # Sales history storage
│── .gitignore        # Ignored files
└── README.md         # Documentation


🛡️ Best Practices
- Keep products.json and sales.json backed up regularly.
- Use .gitignore to avoid committing sensitive files like .env.
- Extend with a database (e.g., MySQL) for production use.
- Follow semantic commit messages for clarity in version control.
